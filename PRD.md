* ## 需求文档 (PRD)

  ### 1. 项目概述

  **项目名称**：西南交通大学出版社AI赋能平台
  **定位**：面向轨道交通领域的智能化知识服务平台，体现"AI+轨道交通+出版"的融合创新

  ### 2. 核心需求

  - **品牌识别**：突出西南交通大学轨道交通学科优势
  - **AI赋能概念**：展示人工智能技术在知识服务中的应用
  - **极简大气**：参考附件图片的深色科技感风格
  ![](https://raw.githubusercontent.com/petal111/PicGo/main/img/20260210101531646.png)
  ![](https://raw.githubusercontent.com/petal111/PicGo/main/img/img_v3_02up_72de671e-0ec2-49c7-8f78-3c505632b95g.jpg)
  - **主题色**：#0A87E2（科技蓝）

  ### 3. 页面结构

  1. **导航栏**：Logo + 导航菜单 + 登录入口
  2. **Hero区域**：全屏高铁动态背景 + 核心标语 + CTA按钮
  3. **AI能力展示**：四大AI赋能模块（智能检索/知识图谱/智能推荐/数字人）
  4. **数据可视化**：动态数据展示（知识库规模/服务数据）
  5. **技术架构**：AI技术栈展示
  6. **页脚**：版权信息 + 联系方式

  ### 4. 设计规范

  - **主色调**：#0A87E2（科技蓝）
  - **背景色**：深色渐变（#0a0e27 → #1a1f3a）
  - **字体**：思源黑体 / Inter
  - **动效**：流畅的滚动动画 + 微交互
